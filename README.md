# 🐔 ChickenCounter Pro-Edition

¡La herramienta definitiva para el conteo avícola moderno! No más dudas sobre cuántos pollos tienes en el corral. Con una interfaz intuitiva y un modelo de negocio innovador, esta calculadora redefine la gestión de granjas.

## ✨ Características Principalas

* ➕ **Suma unitaria (+1):** Para cuando nace un pollito.
* ➖ **Resta unitaria (-1):** Por si alguno decide irse de aventura.
* 🚀 **Suma por lote (+10):** Optimización máxima para granjas industriales.
* 🔒 **Función de Reinicio (Bloqueado):** Seguridad de datos de nivel empresarial. Para limpiar el contador, se requiere la **Versión PRO**.
